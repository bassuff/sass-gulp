### Node.js installieren
```sh
$ yaourt -S nodejs --noconfirm
```
### npm installieren / aktualisieren
```sh
$ yaourt -S npm --noconfirm
# npm install npm -g
```
### package.json erstellen
```sh
$ npm init
```
### Gulp installieren
```sh
$ npm install --global gulp
$ npm install --save-dev gulp
```
### Sass installieren
```sh
$ npm install --save-dev gulp-sass
```
